# Evershop E-commerce Platform - Test Cases

## Overview
This repository contains **full functional and API test cases** for an Evershop-based e-commerce platform.  
It covers everything from product search, filters, cart, checkout, user authentication, to API validations.

Evershop is a modular, Node.js-based platform designed for small to medium online stores.

---

## Key Features
- Product catalog with categories, filters, and sorting  
- Product search with autocomplete and typo correction  
- Shopping cart and wishlist  
- Checkout with multiple payment methods (COD, Stripe, PayPal)  
- User registration, login, and password reset  
- Order management and tracking  
- API endpoints for products, orders, users  

---

## Test Cases
All test cases include:
- Test Case ID  
- Test Scenario  
- Priority  
- Precondition  
- Test Steps  
- Test Data  
- Expected Result  
- Actual Result  
- Status  

Modules Covered
1. Authentication (Login, Registration, Password Reset)  
2. Product Listing and Details  
3. Cart & Wishlist  
4. Checkout & Payment  
5. Search (full-text, partial, typo, autocomplete, out-of-stock handling)  
6. Filters & Sorting  
7. API Testing (Products, Orders, Users)
